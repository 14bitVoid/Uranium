Detailed Description

Container object for the scene graph.

The main purpose of this class is to provide the root SceneNode.

Member Function Documentation

def UM.Scene.Scene.Scene.acquireLock (   self)

Acquire the global scene lock.

This will prevent any read or write actions on the scene from other threads, assuming those threads also properly acquire the lock. Most notably, this prevents the rendering thread from rendering the scene while it is changing.
